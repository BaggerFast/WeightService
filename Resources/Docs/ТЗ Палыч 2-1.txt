Техническое Задание
на программу настройки весовых платформ
Палыч 2.1

Требования:
1. Программа (ПО) должна представлять из себя web-приложение
2. ПО должно обеспечивать windows авторизацию
3. ПО должна использовать ORM NHibernate для работы с БД ScaleDB
4. Для повышения гибкости настройки программы при дальнейшей эксплуатации необходимо задействовать в классах сущностей таблицы db_scales.AttributeDefinationList и db_scales.AttributeValues. Точнее, описать работу с ними в базовом абстактном классе BaseEntity.cs

Сущности, которые необходимо отобразить в ПО:

1. Номенклатура (Единицы измерения (в том числе специальные для контрагентов), Штрихкода, Дополнительные свойства):
	GetList; Detail;Create;Update;Delete;
	Сущность редактируется вручную, может загружается из внешней системы.
   
2. Производственные площадки
	GetList; Detail;Create;Update;Delete;
	Сущность редактируется вручную, может загружается из внешней системы.
	
3. Цеха (склады)
	GetList; Detail;Create;Update;Delete;
	Сущность редактируется вручную, может загружается из внешней системы.
	
4. Весы
	GetList; Detail;Create;Update;Delete;
	Редактиуется только в ПО
	На основании получаемых данных из таблиц db_scales.WeithingFact и db_scales.ProductSeries
	на форме детализации необходимо отбразить текущее состояние.
	Выбранную номенклатуру, текущую серию, среднее кол-во взвешиваний за получасовой интервал.
	
5. PLU
	GetList; Detail;Create;Update;Delete;
	Редактиуется только в ПО.
	Свойства заполняются на основании выбраной номенклатуры, единицы изменения.

6. Шаблоны термочеков 
	GetList; Detail;Create;Update;Delete;
	Редактиуется только в ПО

7. Настройка дополнительных атрибутов у выше перчисленных сущностей.
	GetList отфильтрованный по имени сущности; Detail;Create;Update;Delete;
	Нужно перечисление списка сущностей для фильтации или таблица в БД.
	Необходимо провести исследовательскую работу на тему XSLT преобразований для получения значений реквизисов PLU, хранимых таким способом для печати значений реквизитов на термочеке.
	
8. Нужна форма управления заданиями на взвешивание с указанием PLU, количества, шаблона термочека, времени начала/окончания выполнения задания. Таблицы 
	db_scales.Orders, db_scales.OrderStatus
